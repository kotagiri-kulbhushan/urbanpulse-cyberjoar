REST API Endpoints
GET /api/zones All zones (JSON)
GET /api/zones/<area> Single zone detail
GET /api/summary KPI summary stats
GET /api/top?n=5 Top-N zones by GVS
GET /api/tier/critical Filter by tier
POST /upload/csv Upload CSV file
POST /upload/json Upload JSON array
GET /?pricing=0.40&rental=0.30 Custom weights
All GET endpoints return {"status":"ok","data":[...]}
